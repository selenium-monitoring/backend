= API Documentation

== Ping Endpoint

This endpoint is used to check the availability of the API.

=== Request

[horizontal]
* Method: `GET`
* URL: `/api/ping`

=== Response

* Status Code: `204 No Content`
* Description: Indicates the API is up and running.
